1. Wymienić oraz opisz dwie części selektora

    • Budowa selektora
      To część CSS, która określa, do których elementów HTML będą stosowane reguły stylizacji. Dzięki selektorowi możemy wskazać jeden element, grupę elementów lub elementy spełniające określone warunki.
      np. Selektor typu: p, h1. Selektor klasy: .menu . Selektor ID: #menu.

    • Deklaracja (Declaration Block):
      To część umieszczona w nawiasach klamrowych {} po selektorze. Składa się z jednej lub wielu właściwości CSS (property) oraz odpowiadających im wartości (value).
      np.
      p {
      color: red; /* kolor tekstu */
      font-size: 14px; /* rozmiar czcionki */
      }
