definiowanie koloru znacznika p
Zmiana kolor liter → sposobem pierwszym z tabeli/instrukcja
100 + numer miesiąca urodzenia = 100 + 4 = (104)₁₀ = (#68)₁₆
150 + numer dnia urodzenia = 150 + 30 = (180)₁₀ = (#B4)₁₆
50 + numer buta = 50 + 24 = (74)₁₀ = (#4A)₁₆
czyli kolor w HEXA #68B44A dla znacznika p
wypisz: znacznik p → {color:#68B44A;} wielkość liter 10+N = 10+9 = 19mm
==============================
definiowanie koloru znacznika body
Zmiana kolor liter → sposobem czwartym z tabeli/instrukcja
R = 125 + numer dnia urodzenia = 125 + 30 = 155 → 155/255×100% = 60.78% ≈ 61%
G = 50 + numer buta = 50 + 24 = 74 → 74/255×100% = 29.01% ≈ 29%
B = 200 + numer miesiąca urodzenia = 200 + 4 = 204 → 204/255×100% = 80% ≈ 80%
czyli kolor w RGB 61%, 29%, 80% dla znacznika body
znacznik body → {color: rgb(61%,29%,80%);} wielkość liter 10+J = 10+6 = 16mm
==============================
